Architecture

  • Client (HTML/CSS/JS) in app/cli
  • Server (PHP procedural) in app/srv
  • Data file: onlineOrders/onlineOrders.db (one line per order)
  • HTTP: GET for reading, POST for writing
  • Security: local HTTPS (Apache SSL)

Features

  1. Presentation (home + this page)
  2. Create Flower Order (POST → save to file)
  3. List All Orders (server generates simple HTML table)
  4. Get Order by Code (GET)
