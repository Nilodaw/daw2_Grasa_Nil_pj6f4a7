Floristeria Bloom

Simple PHP project for managing flower orders online

Project Overview

This web application lets customers place, list, and check flower orders. It uses a client/server split: HTML/CSS/JS on the client and PHP (procedural) for server logic.

  • Client in app/cli
  • Server logic in app/srv (no server-side HTML except list view)
  • File store: onlineOrders/onlineOrders.db
  • GET for reading, POST for writing, HTTPS enabled
